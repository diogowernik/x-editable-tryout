# Demo for [bootstrap-x-editable-rails](https://github.com/klenis/bootstrap-x-editable-rails)

## Usage

### 1. Clone repository

### 2. Setup project

```
# bundle install
# rake db:migrate
# rake db:seed
```

### 3. Start Rails

```
# rails server
```

### 4. Visit http://localhost:3000/
